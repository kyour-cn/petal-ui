# Petal-UI

> 基于Uniapp & Vite的轻量化UI库

## 主要特性
* 开箱即用
* 基于Vite
* 组合式Api
* 主题定制
* 深色模式

## 快速开始
使用git clone下载示例项目

```bash
git clone https://github.com/kyour-cn/petal-ui-demo.git
```

下载依赖项
```bash
pnpm install
```

运行Demo
```bash
pnpm run dev
```
